Internet-Native Payments for Humans & AI Agents

1 line of code to accept digital dollars. No fee, 2 second settlement, $0.001 minimum payment. Open standard for programmable payments on the internet.

Get Started Read the Docs

It's This Simple

Add payments to your API in seconds, not weeks

app.use(
  // How much you want to charge, and where you want the funds to land
  paymentMiddleware("0xYourAddress", { "/your-endpoint": "$0.01" })
);
// That's it! See examples for complete implementation.

Works with Express, FastAPI, Hono, Next.js, and more

Why X402?

Built for the future of internet commerce

🔓

Open Standard

Never rely on a single party. X402 is an open protocol that anyone can implement, ensuring long-term stability and independence.

⚡

HTTP Native

Seamlessly integrates with existing HTTP infrastructure. No additional requests needed—payment happens in the same flow.

🌐

Chain Agnostic

Support for EVM chains, Solana, Sui, and more. Choose the blockchain that fits your needs without changing your code.

🔒

Trust Minimized

Cryptographic guarantees ensure funds can only move according to client intentions. No middleman risk.

⛽

Gasless

Clients and servers don't worry about gas fees. The facilitator handles all blockchain complexity for you.

🚀

Easy Integration

One line for servers, one function for clients. Abstract away crypto complexity while maintaining security.
